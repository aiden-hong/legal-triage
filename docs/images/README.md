# 스크린샷 이미지

이 폴더에 웹 UI 스크린샷을 추가하세요.

## 필요한 스크린샷

| 파일명 | 설명 | 촬영 방법 |
|--------|------|----------|
| `streamlit-main.png` | 메인 화면 전체 | 앱 실행 후 초기 화면 캡처 |
| `streamlit-input.png` | 입력 영역 | 왼쪽 입력 영역만 캡처 |
| `streamlit-result.png` | 결과 화면 | TYPE_1 결과가 나온 화면 캡처 |
| `streamlit-redflags.png` | Red Flags 섹션 | Red Flags 펼친 상태로 캡처 |
| `streamlit-guardrails.png` | 가드레일 섹션 | 가드레일 목록이 보이게 캡처 |

## 스크린샷 촬영 방법

### Windows
1. `Win + Shift + S` 키로 캡처 도구 실행
2. 원하는 영역 선택
3. 이 폴더에 위 파일명으로 저장

### macOS
1. `Cmd + Shift + 4` 키로 캡처
2. 원하는 영역 드래그
3. 이 폴더에 위 파일명으로 이동

## 권장 사양

- 해상도: 1200px 이상 너비
- 형식: PNG
- 브라우저: Chrome 권장
